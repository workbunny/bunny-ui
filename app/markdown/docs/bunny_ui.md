### 什么是 Bunny UI?

> 一套 HTMX 开发模式的 Web UI 组件库，使用 AJAX、CSS 过渡、WebSockets 和 服务器推送事件，通过 属性 构建 现代用户界面，结合 简单性 和 超文本的强大功能。
